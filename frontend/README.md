# Mesh Studio Frontend (Next.js)

This frontend is the user interface for Mesh Studio.

It lets users prompt the AI, view generated 3D results, and iterate quickly with the backend.

## Prerequisites

- Node.js 20+
- npm

## Quick Start

Run in PowerShell:

```powershell
Set-Location "./frontend"
npm install
npm run dev
```

Frontend URL: `http://localhost:3000`

## Notes

- Make sure the backend is running (default: `http://127.0.0.1:8000`).
- If API calls fail, verify frontend environment values and backend URL configuration.
